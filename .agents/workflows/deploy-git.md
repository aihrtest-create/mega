# Выполнение автоматического деплоя (GitHub -> GitLab Mirror)

Когда пользователь просит "деплой на гит" или запускает этот воркфлоу, выполни следующие шаги, чтобы отправить локальные изменения в репозиторий GitHub (откуда они автоматически скопируются в GitLab и задеплоятся на сервере):

1. Выполни команду `git add .`, чтобы добавить все измененные или новые файлы в индекс.
2. Выполни коммит изменений с помощью команды `git commit -m "Автоматический деплой по запросу"`. 
3. Запушь коммит в ветку main с помощью команды `git push origin main`.
4. Сообщи пользователю об успешном деплое и напомни, что обновления появятся на сайте через минуту-две (после прохождения зеркалирования на GitLab).
